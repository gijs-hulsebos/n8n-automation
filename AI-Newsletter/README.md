## 🔥 Technical Highlights 

* **🛡️ Dual-Model Synthesis Architecture**: Leverages **Gemini 2.0 Flash** for high-speed synthesis and **GPT-4o Mini** as a deterministic fallback, ensuring the newsletter is generated even during API outages.
* **📄 Dynamic PDF Generation Engine**: Orchestrates the Google Docs API to programmatically build a 4-page "Strategic Intelligence Report" featuring automated layouting, futuristic cover art, and clickable source citations.
* **🎨 Daily Visual Identity Rotation**: A custom JavaScript-driven logic injects a fresh, high-resolution futuristic image into every report, ensuring a unique visual experience for every daily edition.
* **🔀 Logic-Based User Routing**: Implements a sophisticated **IF/Switch architecture** that queries **Supabase** to determine a user's delivery preference: High-fidelity PDF attachment (Gmail) or high-speed HTML summary (MailerLite).
* **🔗 Automated Source Validation**: Employs Regex-based cleaning and URL validation to ensure all curated insights are linked to verified sources, maintaining high journalistic integrity.
* **⚙️ Centralized Error Governance**: Integrated with a global **Error Trigger** workflow, providing real-time alerts for any node failures to maintain a 99.9% delivery success rate.

## 🛠️ Architecture & Stack

* **Orchestration**: n8n (Self-hosted/Cloud)
* **AI Models**: Google Gemini 2.0 Flash & OpenAI GPT-4o Mini
* **Database**: Supabase (CRM & Preferences)
* **Content Processing**: Node.js/JavaScript for dynamic image injection & text cleaning
* **Integrations**: 
    * **Google Docs & Drive API**: For document assembly and PDF rendering.
    * **MailerLite**: For professional subscriber management and bulk HTML delivery.
    * **Gmail API**: For personalized PDF delivery to "Pro" subscribers.
    * **Supabase API**: Serving as the central Source of Truth for user data.

This workflow follows **GDPR best practices** by utilizing MailerLite’s automated unsubscribe management. All credentials are encrypted via n8n’s secure vault. No user data from Supabase is ever used to train AI models; data processing is strictly ephemeral.
